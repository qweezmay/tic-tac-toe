Игра "Крестики-нолики" (Tic-Tac-Toe)

Описание:

Классическая консольная игра на C++ для двух игроков. Игроки по очереди ставят X и O на поле 3×3. Цель игры — первым составить линию из трех своих символов по горизонтали, вертикали или диагонали.

Возможности:

```Двухпользовательская игра в терминале
```Визуализация игрового поля с текущим состоянием
```Проверка победы по строкам, столбцам и диагоналям
```Определение ничьей при полном заполнении поля
```Защита от некорректных ходов

Особенности реализации:

```Использование массива для хранения состояния игрового поля
```Функции для отображения поля, проверки условий победы и обработки ходов
```Циклическая структура для поочередных ходов игроков

1. Установка и запуск, установка компилятора C++ (G++)

```Откройте Терминал (Ctrl+Alt+T)
```Обновите список пакетов и установите компилятор:
```sudo apt update && sudo apt install build-essential
   
2. Проверьте установку:
```g++ --version
3. Получение проекта с GitHub

```Установите Git (если еще не установлен):
```sudo apt install git
```Клонируйте репозиторий:
```git clone https://github.com/qweezmay/tic-tac-toe.git
```Перейдите в папку проекта:
```cd tic-tac-toe

4. Компиляция и запуск программы
```Скомпилируйте C++ файл:
```g++ main.cpp -o tic_tac_toe
   
5. Запустите игру:
```./tic_tac_toe
   
Правила игры

1. Игрок X ходит первым
2. Игроки по очереди выбирают клетку для своего хода
3. Для хода введите координаты (например, 1 1 для верхней левой клетки)
4. Побеждает игрок, собравший три символа подряд
5. Если все клетки заняты и победителя нет — объявляется ничья
